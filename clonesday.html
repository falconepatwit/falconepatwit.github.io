Clones Day Adventure,
Boss Rush Platformer Game based on Datastructures

Home

Java, JavaFX


This group project was created to help us understand data structures and sorting better by developing a boss rush-style game and displaying these components visually. The game consisted of four different levels: queues, stacks, binary trees, and the sorting algorithm Merge Sort. Our menu GUI would allow users to select each stage through unique key presses. The stack level shows off stacks through the "Burger Boss", who throws condiment attacks at the player that are organized and refreshed in a stack format. The queue boss has different attacks that it queues up, then unleashes in a queue-like order at the player. The binary tree level gave the player a prompt of binary tree traversal methods, and the player must go to the correct portion of the map represented by that particular prompt or be attacked by the tree boss. Finally, the Merge Sort level consisted of a boss with multiple spheres, which were random at first, sorted using Merge Sort, then used to attack the player. This project was created using numerous organized classes and helper methods in Java, allowing for easier distribution of work. In addition, we used GitHub for source control, and to allow us to each work in our branch, at our own pace.
